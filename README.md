# Advanced Wealth Tracker— დავითი ამირანაშვილი

## K კოეფიციენტი

K კოეფიციენტი გამოითვლება შემდეგი ფორმულით:

```
K = (სახელის ასოების რაოდენობა + გვარის ასოების რაოდენობა) / დაბადების რიცხვი
```

### ჩემი მონაცემები

| პარამეტრი | მნიშვნელობა |
|---|---|
| სახელი | დავითი |
| სახელის ასოები | დ-ა-ვ-ი-თ-ი = **6** |
| გვარი | ამირანაშვილი |
| გვარის ასოები | ა-მ-ი-რ-ა-ნ-ა-შ-ვ-ი-ლ-ი = **12** |
| დაბადების რიცხვი | **24** |

### გამოთვლა

```
K = (6 + 12) / 24 = 18 / 24 = 0.75
```

### Final Savings ფორმულა

```
Final Savings = (Income - Expenses) * K
Final Savings = (Income - Expenses) * 0.75
```
